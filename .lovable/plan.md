## Проблема

Сейчас в шапке три ссылки — **Проблема (#problem)**, **Решение (#solution)**, **Контакт (#contact)** — и кнопка **Записаться** (#contact). Ни один из этих якорей на странице не существует. После реструктуризации лендинга реальные секции имеют id: `top`, `lead-top`, `amplitude`, `method`, `testimonial`, `research`, `lead-bottom`. Поэтому клики по навигации сейчас никуда не ведут.

## Решение

Переписать навигацию в `src/components/landing/SiteHeader.tsx` так, чтобы пункты соответствовали реальной структуре страницы и помогали пользователю быстро попасть в ключевые блоки.

### Новый набор пунктов навигации

| Пункт              | Якорь          | Куда ведёт                         |
|--------------------|----------------|------------------------------------|
| Метод              | `#method`      | Преимущества метода (7 пунктов)    |
| Амплитуда          | `#amplitude`   | Блок «Хаос → Амплитуда»            |
| Отзыв              | `#testimonial` | Цитата Максима Черкова             |
| Исследования       | `#research`    | Научные статьи                     |

CTA-кнопка справа: **Записаться** → `#lead-top` (поднимает к верхней форме заявки, а не к несуществующему `#contact`).

### Поведение

- На мобильных (< md) пункты навигации остаются скрытыми, как сейчас; видны только логотип и CTA. CTA на мобильных тоже показываем (сейчас он `hidden md:inline-flex` — изменим на видимый, чтобы основная конверсионная кнопка была доступна с телефона).
- Плавная прокрутка к якорям уже работает глобально через `scroll-behavior: smooth` (если нет — добавим в `src/styles.css` к `html`).

### Затрагиваемые файлы

- `src/components/landing/SiteHeader.tsx` — обновить ссылки nav и href CTA.
- `src/styles.css` — при необходимости добавить `html { scroll-behavior: smooth; }` и небольшой `scroll-margin-top` секциям, чтобы якоря не уезжали под фиксированную шапку.

## Открытые вопросы

1. Оставить 4 пункта навигации (Метод / Амплитуда / Отзыв / Исследования) или сократить до 3 самых важных (Метод / Исследования / Отзыв)? По умолчанию — 4 пункта.
2. Показывать CTA «Записаться» на мобильных? По умолчанию — да.
